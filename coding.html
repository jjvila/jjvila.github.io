ShareSquare App, 2018

I was employed at ShareSquare, LLC at the beginning of 2018 with tasks associated with Android development and web design. The main objective in source code development was to re-develop an existing application and add features according to specifications. My focus was the UI/UX component of the app with occasional work at the server side. Among the tasks were optimization, drawing iconography, and implementing new features to improve the user experience.

Like most of my web projects, I re-designed the application’s website from scratch. You may visit the current site here, though the content may have changed. For an archival version representative of my work, contact me and I will follow up accordingly.

Wikimedia Contributions, 2008 - 2012

My journey into Javascript began while I was a Wikimedia contributor prior to attending college.

A notable script was called “licensereviewer,” originally written by another Wikimedia user but rewritten and extended by myself in 2010. The regex-intensive script was designed for users at Wikimedia Commons to quickly perform actions on image pages that needed copyright reviews. According to its history log, it became obsolete in 2017 after seven years of operation. You may view the source code here.
